# AI Test Automation Demo

## Overview
This project demonstrates Selenium automation with AI-assisted failure analysis using pytest.

## Features
- UI automation using Selenium
- Pytest-based framework
- AI-based failure suggestions (simulated)
- Unit testing for AI logic
- HTML test reporting

## Test Scenario
- Automates login functionality
- Validates successful login
- Provides AI suggestions on failure

## AI Integration
AI is used to analyze errors and suggest fixes:
- Locator issues
- Timeout issues
- General debugging suggestions

## Future Enhancements
- OpenAI / LLM integration
- Screenshot capture on failure
- GitHub Actions CI/CD pipeline
- Page Object Model implementation
- Self-healing locator strategy
